# Bandit Level 0 -> 1

The task in this level is to simply connect to `bandit.labs.overthewire.org`, with user `bandit0`, on port `2220`. The password is `bandit0`.

To do this, we use the `ssh` command.

## Solution

`ssh -p 2220 bandit0@bandit.labs.overthewire.org`.
