# Bandit Level 0

## Overview

This is the beginning of the Bandit Wargame.

The task in this level is to simply connect to `bandit.labs.overthewire.org`, with user `bandit0`, on port `2220`. The password is `bandit0`. Then, we display the contents of a file named `readme`.

To do this, we use the `ssh` command.

## Solution
